A mock **Event Registration** project for practicing building API with Golang, Gin framework and SQL

Following routes are implemented :- 
1. GET("/events")
2. GET("/events/:id")
3. POST("/events")
4. PUT("/events/:id")
5. DELETE("/events/:id")
6. POST("/signup")
7. POST("/login")
